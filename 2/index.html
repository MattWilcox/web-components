Local Time

This component takes a UTC timestamp and displays it inline as the visitors local time. Internally is it using JavaScript's toLocaleString method and the display options use DateTimeFormat: .

The following settings can be configured:

  • utc-timestamp - The UTC timestamp to display in standard format. e.g., 2022-01-20T10:00:00Z
  • time-style - full, long, medium, short. No attribute will omit the time from the output.
  • date-style - full, long, medium, short. No attribute will omit the time from the output.
